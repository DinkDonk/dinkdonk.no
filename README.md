dinkdonk.no
===========

The dinkdonk.no website.

### Installing

```bash
$ yarn install
```

### Building

```bash
$ yarn run build
```

### Local serve and watch

```bash
$ yarn run serve
```

### Create a new project

```bash
$ yarn run create-project [project_name]
```

### Deploying

```bash
$ yarn run deploy
```
